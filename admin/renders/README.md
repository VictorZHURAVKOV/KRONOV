# Библиотека рендеров беседок КРОНОВЪ

**Источник:** ТЗ 7.2.

## Что должно быть

11 моделей × 6 ракурсов = **66 PNG-файлов**.

Структура папок:
```
kr-001/
  front.png
  34-left.png
  34-right.png
  back.png
  birdeye.png
  night.png
kr-002/ …
...
kr-011/
```

## Требования к PNG

- **Альфа-канал** (прозрачный фон)
- **≥2000×2000 px**
- **Естественные тени** под беседкой (солнце в зените)
- **Без окружения** (травы, цветов, земли) — только беседка
- **Цветовая точность** — соответствие палитре КРОНОВЪ

## Кто делает

3D-дизайнер (отдельный подрядчик, не входит в scope сайта).

## Как проверить готовность

```bash
# В корне admin/renders/
find . -name "*.png" | wc -l
# Должно быть 66
```
